About Crypto Primitive Detector

An advanced AI/ML-powered web tool designed for identifying cryptographic algorithms and primitives in multi-architecture firmware binaries.

Perfect for reverse engineers, IoT security researchers, and firmware analysts.

Start Analyzing View on GitHub

Rule-Based Detection

Matches known constants like AES S-box, SHA round constants, MD5 tables.

ML Classification

Disassembles code and uses Random Forest to flag crypto-like functions.

Multi-Architecture

Supports ARM, ARM64, MIPS32, x86_64 with Capstone disassembly.

Built with Flask • Bootstrap • Chart.js • Capstone • scikit-learn
